## AI Methodology Disclaimer

This case study package was developed with AI-assisted drafting and human-directed source validation.

- **Source-grounding rule:** Quantitative claims and quotes map to cited sources in Additional-Sources exhibits
- **Verification rule:** Unsupported claims are excluded; partially supported claims are caveated
- **Interpretation limit:** Analytical framing is instructional (not evidence) unless tied to a source
- **Publication control:** Unresolved precision risks must be reviewed in verification-debt.yaml before final publication

All documents include detailed disclaimers about AI involvement and source methodology.

## Recommended Workflow

### For Instructors Using This Case

**Week 1 (Preparation):**
1. Read the Main Case (7 pages, ~30 min)
2. Review the Teaching Note introduction and learning objectives (5 pages)
3. Skim the Supplement for glossary and frameworks (4 pages)

**Week 2 (Class Prep):**
1. Read the full Teaching Note discussion guide (19 pages)
2. Prepare your board plan based on the suggested outline
3. Print the case for student handouts

**Class Day:**
1. Distribute case handout (Unilever_Case.pdf)
2. Follow discussion guide timing: Opening (5-7 min) → Sections 1-6 (60-70 min) → Closing (8-10 min)
3. Use board plan to track student reasoning

**Post-Class:**
1. Collect student responses to discussion questions
2. Use assessment rubric to grade write-ups or presentations
3. Gather feedback for refinement
